# Queries

Queries are transformed into GET requests.

Given the following example:

```graphql
type Comment {
  id: ID
  text: String
}

type Query {
  comments: [Comment]
}
```

Sofa will create a `GET /comments` route.

```bash
curl http://localhost:3000/api/comments
```
